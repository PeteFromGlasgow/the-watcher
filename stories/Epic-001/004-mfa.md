Story 4: Multi-Factor Authentication (MFA)
------------------------------------------

**Title:** User can enable TOTP for enhanced security

> **As a** security-conscious user,**I want** to enable Time-based One-Time Password (TOTP) multi-factor authentication,**so that** I can add an extra layer of protection to my financial account.

### Acceptance Criteria

*   A "Security" section exists in the user's account settings page.
*   The user can initiate a flow to set up MFA.
*   The Nuxt app displays a QR code and setup key provided by Ory Kratos.
*   The user must confirm setup by entering a valid code from their authenticator app.
*   Once enabled, the login flow will require the second factor.
*   The user has the ability to disable MFA from their settings page.
